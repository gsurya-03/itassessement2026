# Final IT Project

This project is a front-end based application developed as part of an academic IT project. It demonstrates the use of modern web technologies to build a clean, responsive, and structured user interface.

---

## Overview

The project focuses on designing a visually appealing interface by combining custom CSS with Tailwind CSS for flexibility and efficient styling, along with JavaScript for interactivity.

---

## Technologies Used

- HTML5  
- CSS3  
- Tailwind CSS  
- JavaScript  

---

## Features

- Responsive design using Tailwind CSS  
- Custom styling with CSS for fine control  
- Clean and structured UI layout  
- Interactive elements using JavaScript  
- Maintainable and scalable code structure
